Prototype

Prototype是什么？Prototype存在于Object还是Function

	Prototype是函数所具有的，
	证明如下：
		1.先看Object对象的prototype和__proto__打印：
		let json = {

		}
		console.log(json.prototype);		// undefined
		console.log(json.__proto__)			// No undefined


		2.再看Function的prototype和__proto__打印：
		let fun = function(x) {
			this.x = x;		// this指向调用者，所有x会被挂载在调用者身上
		}
		console.log(fun.prototype);			// No undefined
		console.log(fun.__proto__);			// No undefined

Prototype的作用是什么？

	用于函数的实例化作用：
	prototype的作用是在实例化函数成对象的时候，对象的__proto__指向的是函数的prototype，从而实现了实例对象的功能
	console.log(_funA.__proto__ === fun.prototype);			// true
